EJECUCIÓN DE EXPERIMENTOS

Durante el proyecto se realizan un total de 7 experimentos.

Cuando se ejecuta el documento main, lo primero que hace es preguntar-te si quieres realizar
algun experimento. Para ejecutar un experimento se le tiene que contestar el número de experimento.

Veamos cada experimento individualmente:

EXP1: 

Este experimento tiene configurados los valores, pero para realizar-lo no encontramos otra manera que ir
comentando y descomentando el codigo para quitar y poner operadores. 

EXP2: 

Si al ejecutar el main le das a experimento 2, ya se configuraran todos los valores.
Lo único que te pide es que seleciones la funcion generación que quieras usar. Para realizar el 
experimento lo hemos ejecutado tres veces, una para cada función de solución inicial.  

EXP3:

Para realizar el experimento se tiene que ejecutar el codigo tantas veces como quieras, y cada
vez después de seleccionar el experimento 3, solo se tiene que escoger el número de k y lambda que 
queremos usar para el Simulated Annealing.

EXP4:

Para realizar el experimento solo se tiene que ejecutar el codigo tantas veces como quieras, y cada
vez después de seleccionar el experimento 4, solo se tiene que escoger el número de estaciones que 
queremos usar. 

EXP5:

Para realizar este experimento solo se tiene que ejecutar el codigo 4 veces, cada una te preguntará 
que función heurística quieres usar y que algoritmo. 

EXP6:

Para realizar el experimento solo se tiene que ejecutar el codigo tantas veces como quieras, y cada
vez después de seleccionar el experimento 6, solo se tiene que escoger el número de furgonetas que 
queremos usar.

EXP7:

Podemos tener ya el resultado de este experimento ejecutando una vez el código y seleccionando el experimento 7

Adicionalmente, la creación de tablas y gráficos usadas para el trabajo se muestran en un documento aparte (graficas_experimentos.ipynbs)
